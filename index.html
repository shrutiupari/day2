Javascript Tutorial

JavaScript is a scripting language developed by Netscape. It is similar in capability to Microsoft'sVisual Basic, the UNIX-derived Perl, and IBM's REXX. In general, script languages are easier and faster to code in than the more structured and compiled languages such as C and C++. Script languages generally take longer to process than compiled languages, but are very useful for shorter programs.
